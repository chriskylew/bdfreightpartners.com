# Freight Partner Recruitment Funnel ## Overview This project is a recruitment and lead intake funnel for freight partners. It supports applicant pre-qualification, full application submission, follow-up workflows, and compliance pages required for messaging and contact operations. ## Tech Stack - Netlify for hosting and serverless functions - Twilio for SMS campaign and messaging workflows - Resend for transactional email delivery - Cloudflare for DNS, security, and performance ## Features - Pre-qualification flow - Full application flow - Thank-you and redirect handling - Privacy Policy and Terms pages for compliance support - Netlify Functions for backend form processing ## Current Status - Twilio campaign submitted for review - Compliance pages are live and working - Recruitment funnel is deployed - Repository is being prepared for system integration and recruiter review ## Next Steps - Move prequal and application flows into the Lanetool system - Add lead tracking and status handling - Improve validation and modular structure - Expand compliance content as needed ## Author Christopher Wallace
