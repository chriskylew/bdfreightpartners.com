# B&D Freight Partners Static Website

Small Twilio A2P-verifiable static site for B&D Freight Partners.

## Required live URLs for Twilio

Privacy Policy URL:

```text
https://bdfreightpartners.com/privacy-policy.html
```

Terms & Conditions URL:

```text
https://bdfreightpartners.com/terms.html
```

## Twilio consent text

```text
Drivers and owner-operators opt in to receive SMS messages by submitting their phone number through the application and onboarding forms on https://bdfreightpartners.com. During submission, users are informed they consent to receive SMS communications related to application updates, onboarding, scheduling, and operational coordination from B&D Freight Partners. Message frequency varies. Message and data rates may apply. Reply STOP to opt out or HELP for support.
```

## Deploy steps

1. Copy these files into the GitHub repo root.
2. Commit and push to main.
3. Deploy through Netlify, Cloudflare Pages, GitHub Pages, or your current host.
4. Verify these URLs are public:
   - https://bdfreightpartners.com
   - https://bdfreightpartners.com/privacy-policy.html
   - https://bdfreightpartners.com/terms.html
5. Resubmit the Twilio A2P campaign.
